Building a "Block Blast" clone for the web is a great project for a frontend developer, especially using React and Zustand for state management. Since the game relies on a grid and drag-and-drop mechanics, you need a prompt that focuses on the core logic: grid validation, line clearing, and shape generation.

Here is a comprehensive prompt you can use with an AI code generator (like Claude, GPT-4, or V0) to scaffold the application:

The Prompt
"Act as an expert React developer. Build a web-based clone of the game 'Block Blast' using React, Tailwind CSS, and Zustand for state management.

Core Requirements:

The Grid: Create an 8x8 game board. Each cell should be a square with a subtle border.

Shape Generation:

Generate 3 random shapes (tetromino-style blocks) at the bottom of the screen in a 'dock' area.

Shapes should only be replenished once all 3 current shapes are placed on the board.

Drag and Drop Logic:

Implement a smooth drag-and-drop system (using react-dnd or native pointer events).

Blocks must 'snap' to the 8x8 grid.

Validation: A block can only be dropped if the underlying cells are empty.

Game Rules:

When a full horizontal row or vertical column is filled, clear those cells immediately.

Implement a scoring system: Points for placing blocks and bonus points for clearing multiple lines at once (combos).

Game Over: Detect when none of the remaining shapes in the dock can fit anywhere on the board.

Visuals & UI:

Use a Monochrome (Black and White) high-contrast theme.

Use Ant Design icons for the UI elements (reset, sound, trophy).

Add a 'Ghost' preview showing where the block will land while dragging.

Code Structure:

Use Zustand to manage the grid state, currentShapes in the dock, and the score.

Ensure the code is modular, with separate components for Grid, Cell, and Shape."
